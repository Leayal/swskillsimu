SoulWorker Skill Simulator

The skill simulator is made (mainly) based on the information from SoulWorker in Japan region. This skill simulator require javascript turned on. The skill simulator should work on most of modern browsers. I just tested on: Firefox v62, Chrome v69, Edge.

Opensource

Absolutely Opensource (MIT). Every code.

Updated

We try our best to updated it frequently. Website and the code.

Available on GitHub

You can get the actual code on GitHub.

Skills

Free to put points into any skills. Also able to reset your Skill-Tree.
